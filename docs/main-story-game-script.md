# 《魔法学院：失准星钟》主线游戏剧本

版本：剧情整合稿 v1
适用项目：`magic-academy-rpg`
关联数据：`src/data/npcs.json`、`src/data/dialogues.json`、`docs/npc-character-roster.md`

## 一句话故事

玩家以新生身份进入魔法学院后，发现钟塔的“星钟”正在提前、倒放并吞掉时间。全校记忆、课程、事故名单和人物关系被持续改写。玩家必须通过 31 名 NPC 的证词，把散落在图书馆、中庭、大厅、温室、训练场、食堂、草坪、镜湖与月石洞窟的线索拼成完整路径，进入钟塔底层，让被封住的“真正午夜”重新响起。

## 主题

本作主线不是单纯击败反派，而是让玩家判断：真实是否一定比安稳更重要，记忆被修正后的幸福是否仍然算幸福。每个 NPC 都握有一段真相，但他们也都有不愿公开的个人代价。玩家推进剧情的过程，就是让这些人决定是否愿意把自己的秘密交给同一个答案。

## 核心设定

星钟原本用于校准学院魔力潮汐。多年前，钟塔影社为了掩盖一次午夜事故，把事故发生的一分钟封入星钟副轴。封印让事故名单、目击者、课程表和部分学生记忆被改写。现在副轴开裂，星钟开始从全校借走时间，导致预言提前、信件来自明天、湖面重复过去、梦境写出玩家名字。

玩家是“新来的学生”，但不是随机闯入者。星钟选择玩家，是因为玩家的名字没有被旧封印记录，能够进入那些被改写名单排斥的场景。

## 序章：第十三下钟声

### 场景目标

玩家完成基础移动和交互教学，在星辉图书馆遇见莉娅。午夜前，学院钟塔突然敲响第十三下。所有学生都短暂失神，只有玩家和莉娅记得这一下钟声。

### 场景流程

1. 玩家靠近星辉图书馆，触发莉娅主线对话。
2. 莉娅发现《黄昏星图》的空白页写出玩家名字。
3. 钟塔响起第十三下，图书馆灯火倒流三秒。
4. 玩家前往中庭找米拉验证异常。
5. 米拉的镜面星盘显示：钟塔影子每天提前七秒落下。

### 关键对白样例

莉娅：
“我是莉娅，负责管理星辉图书馆的古籍。你对魔法有兴趣吗？如果你只是想找刺激，钟塔传闻会让你失望；如果你想理解魔法的本质，也许我们可以从一本旧星图开始。”

玩家：
“为什么这页上有我的名字？”

莉娅：
“因为星辉图书馆从不随便选择读者。更糟的是，它选择你的时间，刚好是钟声失准的时候。”

米拉：
“别靠镜子太近。它今天没有照出你的脸，只照出了钟塔的影子。影子提前了七秒，而七秒足够一段记忆被人替换。”

### 章节完成条件

获得主线道具：`黄昏星图空白页`、`提前七秒的镜面记录`。
解锁任务：调查学院记录中是否存在更多时间错位。

## 第一章：名单里不存在的人

### 场景目标

玩家通过阿登、塞琳娜和伊芙琳确认学院文书系统被篡改：事故名单里有一个从未入学却每天被点名的人，月宴座次图存在第十三席，纪律签到表有空白项每天在钟塔门口签到。

### 场景流程

1. 阿登在中庭调查校报旧档，向玩家展示事故名单。
2. 塞琳娜在大厅准备月宴礼仪，发现座次图多出第十三席。
3. 伊芙琳试图没收相关文件，但她也发现纪律记录被修改。
4. 玩家需要在“公开给全校”与“先保护证人”之间做一次轻分支选择。

### 关键互动

| 互动 | 结果 |
| --- | --- |
| 支持阿登公开报道 | 阿登好感提升，伊芙琳警戒提高，后续巡查更严格。 |
| 建议阿登隐藏消息源 | 伊芙琳更早加入调查，阿登需要更多证据才推进报道。 |
| 帮塞琳娜违背礼仪进入封存礼堂 | 获得 `第十三席座次图`。 |

### 关键对白样例

阿登：
“事故名单里有个名字每天被点到，可我查遍入学册、宿舍表和成绩单，都找不到这个人。你明白这意味着什么吗？有人不只是被忘了，是被系统故意留成了空位。”

塞琳娜：
“礼仪要求我按名单排座。可第十三席没有客人，椅背却有体温。若我承认它存在，我就必须承认学院礼仪守护的是一场谎言。”

伊芙琳：
“我可以没收你们的纸条，也可以假装没有看见空白签到项。但如果纪律只保护错误，那纪律本身就已经失效。”

### 章节完成条件

获得主线道具：`事故空名册`、`第十三席座次图`、`空白签到记录`。
主线判断：星钟异常不是自然故障，而是人为封印开始松动。

## 第二章：地基下的旧水道

### 场景目标

玩家从物质层面追查钟塔地基。塔莉娅的星藤根须、艾里奥的蓝色炉火、科林的灰烬银砂和若文的苦味糖药共同证明：封印材料来自钟塔墙砖、禁区土壤和被篡改记忆。

### 场景流程

1. 水晶温室星藤根须主动避开正常土壤，指向旧水道。
2. 食堂炉心火焰在钟声失准时变蓝，艾里奥担心食堂被封。
3. 科林承认灰烬实验曾混入钟塔墙砖银砂。
4. 若文的糖药能让被篡改的记忆出现苦味，玩家用它验证第一批 NPC 证词。

### 关键互动

玩家需要选择谁先公开自己的秘密：

| 选择 | 剧情影响 |
| --- | --- |
| 让塔莉娅承认温室土壤来自禁区 | 旧水道路线提前解锁，但温室老师进入调查状态。 |
| 让科林公开实验事故 | 获得银砂配方，科林好感大幅提升。 |
| 让若文先背锅保护配方主人 | 记忆验证延后，但若文后续会在终章替玩家稳住恐慌学生。 |

### 关键对白样例

塔莉娅：
“星藤不会说谎。它们不朝阳光长，却朝钟塔地基下方伸根。那下面有水，也有不该属于学院的土。”

艾里奥：
“你看见炉火了吗？蓝色不是温度，是它在害怕。火焰认得钟塔墙砖，因为那些砖曾被扔进炉里烧过一次。”

科林：
“我以为银砂只是催化剂。后来朋友受伤，我才明白自己烧掉的不是矿粉，是一小段被封住的时间。”

若文：
“吃下去。如果你的记忆是原本的，它会是薄荷味；如果有人替你写过昨天，它会苦得像咬到墨水。”

### 章节完成条件

获得主线道具：`星藤根须图`、`蓝色炉火灰烬`、`钟塔银砂`、`苦味糖药`。
解锁旧水道入口，但入口被影线封住，需要符文训练场线索。

## 第三章：反写的钥文

### 场景目标

玩家在符文训练场学习处理“影线”：莲汐的断刃可以切开钟声留下的影线，卡西娅能反写封印钥文，优奈的雷光会避开影社成员的影子，布拉姆的碎盾裂痕与钟塔封印一致。

### 场景流程

1. 莲汐被要求在决斗中故意输掉，玩家可选择是否介入。
2. 卡西娅拒绝承认论文来自禁书，直到反写符号与星图空白页吻合。
3. 优奈训练雷光，电弧自动避开无脚步影子。
4. 布拉姆展示碎盾，裂痕在钟声中发光。

### 关键互动

| 互动 | 结果 |
| --- | --- |
| 帮莲汐赢下决斗 | 获得 `断刃影线术`，但贵族学生后续阻挠。 |
| 劝莲汐按要求输掉 | 后续进入大厅更顺利，但莲汐需要额外对话才交出断刃线索。 |
| 让卡西娅署名禁书来源 | 获得完整 `反写钥文`。 |
| 安抚优奈控制雷光 | 终章雷光不会误伤 NPC。 |

### 关键对白样例

莲汐：
“他们说东境剑士只懂蛮力，所以要我输得好看。可影线不是用力砍开的，它要被承认、被命名，然后被切断。”

卡西娅：
“我不喜欢承认自己错，更不喜欢承认自己最好的论文来自禁书。但如果禁忌恰好是真相，我宁愿署名承担它。”

优奈：
“雷光在躲那些影子。你看见了吗？它们站在场上，却没有脚印。我的魔法不是失控，它是在替我们指认谁不该存在。”

布拉姆：
“这面盾碎在三年前，可裂痕今天才开始发光。它不是提醒我保护别人，是提醒我别再躲在保护别人后面。”

### 章节完成条件

获得主线技能：`切影`、`反写钥文`、`雷光辨影`、`碎盾裂纹图`。
旧水道影线解除，主线进入镜湖求证时间数值。

## 第四章：湖底的三次钟声

### 场景目标

玩家在镜湖确认星钟并非随机失准，而是每次从全校借走固定时间。梅斯提供湖底回声，诺艾尔推翻旧决斗判决，君珀计算出每次失准借走三百七十二秒。

### 场景流程

1. 梅斯带玩家靠近镜湖，湖底重复被封印前最后三次钟声。
2. 诺艾尔承认一场旧决斗的双方其实相隔一分钟，不应被判作弊。
3. 君珀用珍珠算术推导：星钟每次修补裂缝都从全校借走三百七十二秒。
4. 玩家把 `提前七秒`、`第十三席`、`空白签到`、`三百七十二秒` 合并成“失准公式”。

### 关键对白样例

梅斯：
“我潜进湖底时，听见同一声钟响了三次。老师说那是回声，可回声不会一次比一次更像求救。”

诺艾尔：
“如果我推翻判决，所有人都会说我不公正。可是维持错误判决，只是让我的名誉替谎言站岗。”

君珀：
“三百七十二秒。听起来像冷冰冰的数字，对吧？但把它除回每一个学生身上，就是他们忘掉的招呼、迟到的道歉和没来得及说出口的名字。”

### 章节完成条件

获得主线道具：`湖底三钟回声`、`被推翻的决斗判决`、`三百七十二秒公式`。
主线揭示：被封住的不是地点，而是一分钟。需要找到进入那一分钟的方法。

## 第五章：未来寄来的信

### 场景目标

草坪与中庭线负责进入钟塔的现实路径。芬恩的信来自明天，赛拉的巡夜记录被改，凯尔的刺篱只在钟塔影子经过时让路，马特奥的旧钥匙在钟声倒放时露出齿纹，莉娜的课表指向顶层空教室，欧文的星兽排列成七芒星。

### 场景流程

1. 芬恩交给玩家一封邮戳来自明天的信，信中只有一句话：“不要相信第一次午夜。”
2. 赛拉承认巡夜记录被人改过，提供无脚步人影路径。
3. 凯尔因禁林契约不能说谎，只能用行动带玩家看刺篱让路。
4. 马特奥在倒放钟声中找回旧钥匙齿纹。
5. 莉娜发现不存在课程都指向钟塔顶层空教室。
6. 欧文放开最胆小的星兽，它们排列成七芒星，标出进入钟塔的时机。

### 关键互动

| 互动 | 结果 |
| --- | --- |
| 拆开芬恩的信 | 提前知道“第一次午夜”是假结局，但芬恩担心命运被改写。 |
| 不拆信，只让芬恩保管 | 终章由芬恩主动读信，触发更高好感对白。 |
| 帮马特奥承认丢过真钥匙 | 获得 `倒放钥匙齿纹`。 |
| 让欧文放开星兽 | 获得 `七芒星时机`。 |

### 关键对白样例

芬恩：
“邮戳是明天，收件人是今天的你。你问我该不该拆？我送过很多信，第一次觉得信封里装的不是纸，是别人的命运。”

赛拉：
“我巡夜从不漏记脚步。可那个人影没有脚步声，记录本却替我写上了‘一切正常’。”

凯尔：
“我不能说谎，所以我不说。你跟着我，看刺篱自己让开。它们只给钟塔影子让路，也只给愿意面对影子的人让路。”

马特奥：
“钥匙没有丢。或者说，我一直希望它没有丢。真正的齿纹只在钟声倒放时出现，因为那时谎言也会退回它出生前。”

莉娜：
“这节课不存在，教室不存在，老师不存在。可课表每天都让学生去那里。时间不是乱了，它在点名。”

欧文：
“最胆小的那只星兽从不带路，除非它知道所有勇敢的路线都是陷阱。今天，我们听它的。”

### 章节完成条件

获得主线道具：`明日信`、`巡夜改写记录`、`刺篱让路证明`、`倒放钥匙齿纹`、`空教室课表`、`七芒星时机`。
解锁钟塔顶层空教室，但入口仍缺少梦境变量。

## 第六章：没有门的梦塔

### 场景目标

玩家进入月石洞窟，确认现实中的钟塔入口并不是最终入口。西奥的梦境中，钟塔没有门，只有玩家名字；塞琳的幽光路标只照亮反方向路，说明真正路径在“被排除的方向”。

### 场景流程

1. 西奥让玩家进入一场共享梦境。
2. 梦中钟塔没有门，塔身写着玩家名字。
3. 玩家尝试用钥匙、符文、星图都无法开门。
4. 塞琳的幽光路标照向反方向，带玩家看见旧守钟人的背影。
5. 旧守钟人只说一句：“名字不在名单里的人，才能把名单带回来。”

### 关键对白样例

西奥：
“我每进入一次梦，现实中的一天就会变模糊。但如果你的名字真的写在梦塔上，我宁愿少记得一天，也不想让所有人永远少记得一分钟。”

塞琳：
“我总是迷路，所以大家以为我不可靠。可有些路标就是故意指错方向的。它们不是让你迷路，是保护你不被正确的路抓住。”

旧守钟人：
“钟塔没有门。门是给名单上的人用的。你不在名单里，所以你不是客人，你是钥匙。”

### 章节完成条件

获得主线变量：`玩家名字是钥匙`、`反方向路标`。
解锁最终准备任务：识别钟塔影社成员表，并冻结虚假时间。

## 第七章：假名与安魂曲

### 场景目标

玩家回到图书馆和大厅，集齐最后一组证词。希拉斯的墨灵写出钟塔影社成员表，伊莉娅的玻璃琴补齐七枚校徽音阶，艾莉娅的安魂曲能冻住虚假时间，娜拉用帷幕重演失踪演出，薇拉发现未来午夜借阅记录。

### 场景流程

1. 希拉斯必须夺回真名，墨灵才会写出假名成员表。
2. 伊莉娅修复玻璃琴，每修好一个音都会忘记一段童年；玩家可选择是否暂停。
3. 艾莉娅承认她记得时间回卷里别人忘掉的请求，愿意在终章唱完安魂曲。
4. 娜拉用幻术重演失踪演出，幕布背面显出钟塔地下入口。
5. 薇拉从夜课档案里发现未来三天后的午夜借阅记录，借阅者名字与玩家相连。

### 关键互动

| 互动 | 结果 |
| --- | --- |
| 帮希拉斯夺回真名 | 获得 `影社假名表`，但墨灵变回普通墨水。 |
| 暂停伊莉娅修琴 | 她保留更多童年记忆，但终章安魂曲稳定性下降。 |
| 继续修琴 | 获得完整 `七徽音阶`，伊莉娅个人代价加重。 |
| 让娜拉重演失踪演出 | 进入终章地下入口。 |
| 相信薇拉的未来借阅记录 | 真结局条件之一。 |

### 关键对白样例

希拉斯：
“墨灵一直叫我假名，因为假名安全。真名会痛，会被偷，也会让你不得不回答自己是谁。好吧，今天我回答。”

伊莉娅：
“我又忘了一件事。可能是一首摇篮曲，也可能是妈妈叫我回家的声音。没关系，玻璃琴记住了。你要替我听完它。”

艾莉娅：
“时间回卷时，所有人都忘了自己求过救。只有我记得。安魂曲不是给死者唱的，是给被迫活在谎言里的人唱的。”

娜拉：
“观众以为戏法的重点是消失。错了，真正的戏法是让所有人相信消失本来就应该发生。”

薇拉：
“借阅簿的墨迹来自未来三天后的午夜。借阅者可能是我，也可能是你。最可怕的是，它写得像我们已经同意了结局。”

### 章节完成条件

获得终章道具：`影社假名表`、`七徽音阶`、`安魂曲末节`、`地下入口幕布图`、`未来午夜借阅记录`。
全员证词汇合，钟塔底层开放。

## 终章：真正的午夜

### 进入条件

必须完成以下证据组：

| 证据组 | 至少需要 |
| --- | --- |
| 文书异常 | 事故空名册、第十三席座次图、空白签到记录 |
| 物质异常 | 星藤根须图、钟塔银砂、苦味糖药 |
| 符文异常 | 切影、反写钥文、碎盾裂纹图 |
| 时间异常 | 湖底三钟回声、三百七十二秒公式、明日信 |
| 路径异常 | 倒放钥匙齿纹、七芒星时机、反方向路标 |
| 真名异常 | 影社假名表、七徽音阶、未来午夜借阅记录 |

### 终章场景流程

1. 玩家在钟塔顶层空教室触发第一次午夜。钟声恢复正常，系统给出“修复成功”的假提示。
2. 若玩家拥有 `明日信` 或相信薇拉记录，可选择“不相信第一次午夜”。
3. 星钟倒放，玩家进入底层钟心。
4. 所有 NPC 的证词以星图坐标形式浮现，每个证词对应一枚钟齿。
5. 影社残留意识出现，辩称封印是为了让学院免于恐慌。
6. 玩家用莉娅星图定位，卡西娅反写钥文解封，莲汐切影，艾莉娅安魂曲冻结虚假时间。
7. 玩家做最终选择。

### 终章关键对白样例

影社残留意识：
“我们没有杀死真相，只是把它放慢了一分钟。学院因此平静了三年。你凭什么说醒来一定比安睡更仁慈？”

莉娅：
“因为平静不能建立在所有人被偷走的一分钟上。我的父亲也许错过，也许失败，但他不该被一本被改写的笔记审判。”

伊芙琳：
“纪律不该保护谎言。”

阿登：
“报道不该只写胜利者允许被记住的事。”

西奥：
“梦会结束。醒来会痛。但痛至少属于自己。”

玩家选择提示：
“星钟等待你的决定。真正的午夜可以响起，也可以再次被推迟。”

## 结局分支

### 真结局：真正的午夜

条件：完成六组证据，并在终章选择公开真相。
结果：星钟敲响真正午夜，被封印的一分钟回到所有人记忆中。学院短暂陷入混乱，但 NPC 会根据好感和支线完成度主动承担证词代价。莉娅公开父亲笔记，阿登发布报道，伊芙琳重订纪律，艾莉娅唱完安魂曲。钟塔不再吞掉时间。

结尾旁白：
“午夜终于完整。它没有让所有人立刻幸福，却让每个人重新拥有自己的昨天。”

### 秩序结局：被管理的真相

条件：证据不足或终章选择只向学院管理层公开。
结果：星钟停止恶化，但事故真相只被少数人知道。部分 NPC 的个人矛盾仍然存在，影社假名表被封存。校园恢复表面秩序，偶尔仍会出现七秒误差。

结尾旁白：
“钟声准了，却仍有一些名字只在纸背面发光。”

### 遗忘结局：再封一分钟

条件：接受影社残留意识的提议。
结果：玩家用自己的名字替代旧封印，所有 NPC 忘记调查经过。学院完全恢复平静，但图书馆《黄昏星图》最后一页留下玩家名字。

结尾旁白：
“没人再迟到，没人再做噩梦。只有一本书每晚翻到最后一页，像在等一个不会回来的人。”

### 破钟结局：时间坍塌

条件：强行破坏星钟但证据链不足。
结果：钟塔停止运转，学院各区域陷入永久时间错位。该结局作为失败结局或二周目警告存在，不建议作为主推路线。

结尾旁白：
“真相没有被释放，只是失去了容器。”

## 开发拆分建议

1. 先把序章、第一章和第二章做成可完成闭环，确保玩家理解“星钟失准”和“证据链”。
2. 第三到第五章使用多区域并行推进，玩家可自由选择 NPC 顺序，但关键证据需汇总到主线日志。
3. 第六章和第七章作为终章门槛，强调梦境、假名、安魂曲和未来借阅记录的戏剧性。
4. 终章采用证据门槛触发不同结局，避免玩家只靠单一 NPC 对话进入真结局。
5. 每个 NPC 的主线对话应拆成多轮短互动，每轮只推进一个明确剧情节点；本剧本文档负责把这些局部推进串成完整主线。
