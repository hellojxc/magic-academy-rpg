# Mature Senpai Reference

Initial adult female NPC concept for the magic academy RPG.

Design notes:

- Adult mature big-sister archetype.
- Height target: 172 cm.
- Normal mature body type with elegant proportions.
- Light academy fantasy outfit with a loose draped camisole top, double
  side-slit skirt, bare legs, gold accents, and Roman-style strappy
  high-heeled sandals.
- Intended as concept art for later Hunyuan3D/Blender source-model work.

Files:

- `concept-sheet.png`: first concept/reference sheet.
- `concept-sheet-v2-normal-body.png`: revised normal-body concept, preferred
  for earlier dark-uniform modeling.
- `concept-sheet-v3-light-slit-skirt.png`: earlier preferred concept with a
  lighter simplified outfit, high side-slit skirt, and no stockings.
- `concept-sheet-v4-short-sleeve-lowcut-double-slit.png`: earlier concept with
  short sleeves, a low-cut top, no hair accessory, bare legs, and
  double side-slit skirt.
- `concept-sheet-v5-high-heels.png`: earlier concept with elegant
  high-heeled shoes instead of boots.
- `concept-sheet-v6-camisole-top.png`: earlier concept with a
  camisole-inspired thin-strap top while keeping high heels and the double
  side-slit skirt.
- `concept-sheet-v7-roman-strappy-heels.png`: current preferred concept for
  Hunyuan3D multi-view generation, with elegant Roman-style strappy high-heeled
  sandals.
- `concept-sheet-v8-loose-elegant-camisole.png`: exploratory concept with a
  looser, softly draped camisole top for a more elegant mature look, while
  keeping the double side-slit skirt and Roman-style strappy high-heeled
  sandals.
- `hunyuan-multiview-v7/`: Hunyuan3D upload-ready multi-view images generated
  from the v7 concept.
